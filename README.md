# Camp Onseyawa 2026 — Staff Orientation

Self-contained HTML slideshow for Camp Onseyawa 2026 staff orientation.
All narration audio is embedded; no external assets required.

**Live:** https://aztecnightmare365.github.io/onseyawa-orientation-2026/

## Features
- 41 slides covering camp policies and procedures
- Voice narration on by default (toggle with the 🔊 button)
- Scroll-gated slides — long slides require scrolling to the bottom before advancing
- Ends with a link to the Google Forms orientation quiz (20 questions)

## Editing
`index.html` is the entire site. Audio is baked in as base64 data URIs.
Regeneration scripts (audio, quiz form, quiz videos) live in the parent working folder, not this repo.
